Referencia de Instrucciones

Formato de Números

  • 0x0F - Hexadecimal
  • 0b1111 - Binario
  • 15 - Decimal
  • 0Fh - Hex con sufijo

Destino

  • W o 0 - Registro W
  • F o 1 - Archivo (File)

Directivas

  • ORG 0x00 - Origen
  • END - Fin del programa

Ejemplos

; Mover literal a W
MOVLW 0x0F
; Mover W a archivo
MOVWF 0x05
; Incrementar archivo
INCF 0x05, F
; Saltar a dirección
GOTO 0x10
